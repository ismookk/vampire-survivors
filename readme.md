# 🧛 Mini Vampire Survivors-like (HTML / CSS / JS)

## 📌 프로젝트 개요

HTML Canvas와 Vanilla JavaScript만을 사용해 구현한 뱀파이어 서바이버 라이크 생존 액션 게임.

- 자동 공격 기반
- 다수의 적을 상대로 제한 시간 생존
- 페이즈 기반 난이도 상승
- 선택형 성장 시스템
- 1.0 버전부터 로비 화면, 보스 등장, 속도 동기화 등 핵심 개선 반영

## 🎮 게임 목표

- 5분 동안 생존
- 1분마다 페이즈 전환
- 점점 강해지는 적의 물량과 패턴을 버티는 것이 목표
- 4분 30초 이후 등장하는 보스 처치 추가 목표

## 🕹 핵심 조작

- 이동: WASD / 방향키
- 공격: 자동 타겟팅
- 레벨업 시: 강화 선택
- 일시정지: 버튼 클릭
